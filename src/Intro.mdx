## Get started

레이아웃 컴포넌트를 편하게 사용할 수 있도록 제작된 패키지입니다

### Install

styled-components를 같이 설치하셔야 사용이 가능합니다

`npm i wus-layout-component styled-components -D`

### how to use?

```tsx
import { Flex } from 'wus-layout-component';

<Flex
  css={css}
  justifyContent={justifyContent}
  direction={direction}
  alignItems={alignItems}
  flexWrap={flexWrap}
>
  <div
    style={{
      width: '100px',
      height: '100px',
      backgroundColor: 'gray',
    }}
  ></div>
</Flex>;
```

### Supporting

해당 패키지는 CommonJs와 ES Modules를 지원합니다.
